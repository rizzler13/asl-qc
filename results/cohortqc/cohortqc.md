# Cohort Quality Control Analysis
This project facilitates comprehensive quality control (QC) analysis across aggregated datasets encompassing diverse population samples.

Validation data was sourced from the OpenNeuro repository, specifically the "Resting State Perfusion in Healthy Aging" dataset (Accession: ds000240).

Note: The visualizations provided demonstrate preliminary outputs from a prototype build and are intended solely for proof-of-concept, not for formal research or clinical application.
